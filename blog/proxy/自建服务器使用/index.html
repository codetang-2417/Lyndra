自建服务器使用

　　会发送给你 移动端 和 电脑端 的订阅链接，分别导入使用即可。

代理软件

安卓/ios

　　使用免费软件 sing-box。

　　ios 需要美区账号才能下载，可以参考：美区账号免费注册。

　　安卓直接在 github 官网 下载。

　　底部导航栏选择 Profiles，New Profile，选择 Remote 类型，在 URL 处填入移动端订阅链接。

​

​

　　在 仪表 中点击 启用 后，可在 组 中选择节点，proxy 组中选择代理节点，仅 us，all 可用，其他的地区为虚假延迟。

​

​

电脑端

　　软件 mihomo-party，windows、linux、mac os 通用。

　　下载链接：https://github.com/xishang0128/mihomo-party/releases

　　注：mihomo-party 在功能稳定后，原作者考公上岸，无法继续开发。目前该软件已经被机场主接手，不确定后续开发是否安全。现在推荐使用原开发者之一 xishang0128 fork 后继续开发的版本，该版本更新较慢。 稳定了一段时间，现在应该可以继续使用。原链接：github.com/mihomo-party-org/mihomo-party/releases。

　　在 订阅管理 导入给的电脑端订阅链接。

​

　　无特殊需求，使用 系统代理 模式即可，该模式可代理浏览器的流量，一般情况下不会自动代理其他系统软件的流量。

　　如果需要某些应用也被代理（例如，某些国外 app），需要开启 虚拟网卡 模式。mihomo-party 安装时，会提示用管理员权限，以开启虚拟网卡。

　　虚拟网卡开启后，会接管系统所有流量。

使用方式

电脑端（Windows、Mac OS、Linux）

　　一般使用 规则 路由模式，对国内外网站分流。如果选择了 全局，则所有网站流量均走代理，国内网站会很慢。

　　按照下图进行设置，即可使用。Windows 的 edge 浏览器可能需要在开启系统代理后，重启一次。如果使用虚拟网卡模式，则不需要。

　　三种模式的区别

  • 规则：流量按照一定的分流规则进行分流，可以实现国内的走直连，国外的走代理服务器。
  • 全局：所有流量都走国外代理服务器，国内的网站访问很慢。
  • 直连：所有流量都走直连，不能访问国外被墙网站。

　　正常情况下选择 规则 路由模式即可。

　　规则 路由模式下，代理软件分流规则：

常用国外网站 -> 代理节点

国内 ip -> 直连

国内常用网站/域名 -> 直连

其余的域名/网站 -> 代理节点

　　这个规则会导致一些小众国内网站走代理，造成无法访问，或者访问慢的情况。这时需要关闭代理，或者在代理软件中，使用 覆写 功能，将该网站分流到 直连。

虚拟网卡和系统代理的区别

　　系统代理模式，是在系统上开一个代理端口，不强制其他应用走代理访问。一般来说，浏览器均会检测系统代理，并通过系统代理上网。因此，如果你只有浏览器访问外网的需求，则系统代理就够用。

　　虚拟网卡模式，接管系统的所有流量，一般是在某些软件不会走系统代理的情况开启。例如在 steam 中访问创意工坊，默认不走系统代理时，可以开启虚拟网卡模式。由于强制所有流量通过代理软件访问网络，需要路由分流规则合理，否则会浪费代理流量。

　　需要说明的是，系统代理和虚拟网卡模式，最终都会通过 路由模式 进行分流，尽可能的将国内流量走直连，不占用代理流量。

　　所以，你会在代理组中看到各种各样的规则，将国内网站和国外网站区分开。国内的走直连，被墙的走节点选择。节点选择中，你需要选中美国节点，或者自动选择。

国内小众网站

　　在使用过程中，如果发现某些国内网站无法访问或者很慢，则是该网站被代理到国外了，需要手动在 覆写 模块添加直连规则。

验证覆写规则生效

　　点击连接选项卡，查看对应的域名是否按照规则走代理或者直连。可以搜索过滤。

移动端

　　移动端使用场景比较少，在需要代理时开启即可，也能分流部分国内网站，常用的软件都可以分流。出现问题，关闭代理即可。

节点选择

　　无论是移动端还是电脑端，仅美国节点可用。且最好选择 首选1，最安全。首选1 无法使用时，再尝试备用节点。

流量

　　为了避免不知情时，下载游戏或者其他大文件走国外代理，限制每一个节点的流量为 50G。但可以无限重置，只是为了防止流量超用。

　　当下载游戏时，可以先看看下载是否是使用的直连，如果走节点，需要关闭代理软件（或者将虚拟网卡、系统代理关掉即可）。因为国内的 steam 等平台有国内服务器加速下载，不需要挂代理，虽然我的分流规则会让 steam 和 epic 的下载走直连，但不能保证后续厂商修改对应的下载网址。而且下载游戏一般需要大量流量，走代理会浪费。

　　正常玩游戏是可以的。

　　每个人可以使用3个节点。如果 首选节点 均使用完（正常使用，而不是下载大型游戏或者偷跑掉），可以找我重置。
